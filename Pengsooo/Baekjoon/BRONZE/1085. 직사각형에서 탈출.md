# 1085. 직사각형에서 탈출



## 문제

한수는 지금 (x, y)에 있다. 직사각형의 왼쪽 아래 꼭짓점은 (0, 0)에 있고, 오른쪽 위 꼭짓점은 (w, h)에 있다. 직사각형의 경계선까지 가는 거리의 최솟값을 구하는 프로그램을 작성하시오.

### 입력

첫째 줄에 x, y, w, h가 주어진다.

### 출력

첫째 줄에 문제의 정답을 출력한다.

### 제한

- 1 ≤ w, h ≤ 1,000
- 1 ≤ x ≤ w-1
- 1 ≤ y ≤ h-1
- x, y, w, h는 정수





## 예제

### 입력1

```python
6 2 10 3
```

### 출력1

```python
1
```





## 코드

```python
x, y, w, h = map(int, input().split())

print(min(min(x, w-x), min(y, h-y)))
```
