# 10872. 팩토리얼



## 문제

0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

### 입력

첫째 줄에 정수 N(0 ≤ N ≤ 12)가 주어진다.

### 출력

첫째 줄에 N!을 출력한다.



## 예제

### 입력1

```python
10
```

### 출력1

```python
3628800
```



### 입력2

```python
0
```

### 출력2

```python
1
```





## 코드

```python
def Factorial(n):
    if n > 0:
        return n * Factorial(n-1)
    else:
        return 1
    
n = int(input())
print(Factorial(n))
```
